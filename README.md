## 🚀 功能特性

- 📊 **实时行情** - 查看股票实时价格和涨跌幅
- 📈 **走势图** - 支持分时、五日、日K三种视图
- 💼 **持仓管理** - 记录买入记录，自动计算盈亏
- 🤖 **智能分析** - AI驱动的股票分析和预警
- 🔔 **交易规则** - 自动提醒买卖时机

## 🛠️ 技术栈

- 前端：React + Vite
- 后端：Node.js + Express
- 数据库：SQLite
- 图表：Lightweight Charts
